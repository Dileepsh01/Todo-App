# 📝 Todo App

A simple and responsive Todo App built using React and Tailwind CSS.

---

## 🚀 Features

- ➕ Add Task  
- ❌ Delete Task  
- 🔁 Reset Tasks  
- ✅ Mark as Completed  
- ⌨️ Add task using Enter key  
- 🎯 Auto focus input field  

---

## 🛠️ Tech Stack

- React.js  
- Tailwind CSS  

---

## 📂 How to Run

```bash
git clone https://github.com/Dileepsh01/Todo-App.git
cd Todo-App
npm install
npm run dev
